### Explication Projet
```
Le projet consiste à créer une page web qui permet de prendre des commandes de repas comme uber eat. Ce projet est codé en vue.js.
```

### Description du projet
```
Sur le site nous avons un acceuil, mon compte et mes commandes.

La page d'acceuil permet de visualiser tout les restaurants. Sur cette page l'utilisateur peut trouver le restaurant qu'il souhaite. 
Pour faciliter la recherche nous avons mis en disposition une barre de recherche qui permet aux utilisateurs de trouver le restaurant plus facilement.

L'utilisateur peut s'enregistrer ou se connecter s'il est déjà inscrit. Une fois connecté l'utilisateur peut chercher un restaurant qu'il souhaite en cliquant sur commander.
Lorsque l'utilisateur choisit le restaurant en cliquant sur le bouton plusieurs informations sont affichés concernant le restaurant.

Lorsque l'utilisateur clique sur mon compte il peut voir tout les informations personnels. Pareil pour mes commandes l'utilisateur peut suivre sa commande et son historiques. 

```
